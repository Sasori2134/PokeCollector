# Pokedex Collector აპლიკაცია

თანამედროვე Android აპლიკაცია პოკემონების იდენტიფიცირებისთვის, შეგროვებისა და მართვისთვის ხელოვნური ინტელექტისა და ოფიციალური პოკემონების მონაცემების გამოყენებით.

## ფუნქციები
- **AI-ზე დაფუძნებული იდენტიფიკაცია:** გამოიყენეთ მოწყობილობის კამერა სურათების ან ობიექტების სკანირებისთვის. აპლიკაცია იყენებს **Gemini AI**-ს (Google Generative AI) კადრში არსებული პოკემონის ამოსაცნობად.
- **ოფიციალური მონაცემების ინტეგრაცია:** იდენტიფიცირების შემდეგ, აპლიკაცია რეალურ დროში იღებს მონაცემებს (Pokedex ID, სიმაღლე, წონა, გამოსახულებები) **PokeAPI**-დან.
- **მომხმარებლის ავტორიზაცია:** უსაფრთხო შესვლისა და რეგისტრაციის სისტემა, რომელიც მუშაობს **Firebase Authentication**-ის საშუალებით.
- **Cloud კოლექცია:** თქვენი დაჭერილი პოკემონები ინახება თქვენს პირად პროფილში **Firebase Firestore**-ის გამოყენებით.
- **ფავორიტების სისტემა:** მონიშნეთ თქვენი საყვარელი პოკემონები მათ სპეციალურ სიაში სანახავად.
- **დინამიური დეშბორდი:** თვალი ადევნეთ თქვენი კოლექციის სტატისტიკას და ნახეთ ბოლოს დაჭერილი პოკემონები მთავარ ეკრანზე.

## ტექნიკური დეტალები
- **არქიტექტურა:** MVVM (Model-ViewModel-Intent) სუფთა არქიტექტურის (Clean Architecture) პრინციპებით.
- **UI Framework:** 100% Jetpack Compose თანამედროვე და რეაქტიული ინტერფეისისთვის.
- **ნავიგაცია:** Jetpack Compose Navigation მორგებული ქვედა ნავიგაციის ზოლით (Bottom Navigation Bar).
- **კამერის სისტემა:** აგებულია **CameraX**-ის გამოყენებით აპარატურულად დაჩქარებული კამერის პრევიუსა და სურათების გადასაღებად.
- **სურათების ჩატვირთვა:** **Coil** ბიბლიოთეკა სურათების ეფექტური ასინქრონული ჩატვირთვისა და ქეშირებისთვის.
- **Networking:** **Retrofit** და **Gson** RESTful API-ებთან ურთიერთქმედებისთვის.
- **მონაცემთა ბაზა:** **Firebase Firestore** (NoSQL) რეალურ დროში მონაცემთა შესანახად.
- **AI Engine:** **Gemini 2.5 Flash** Google AI SDK-ის მეშვეობით Android-ისთვის.

## როგორ გავუშვათ
1. მოახდინეთ რეპოზიტორის კლონირება.
2. დაამატეთ თქვენი `google-services.json` ფაილი `app/` დირექტორიაში.
3. აიღეთ Gemini API გასაღები [Google AI Studio](https://aistudio.google.com/)-დან და დაამატეთ ის `GeminiService.kt` ფაილში.
4. გაააქტიურეთ Email/Password Auth და Firestore თქვენს Firebase კონსოლში.
5. ააწყვეთ და გაუშვით ფიზიკურ Android მოწყობილობაზე ან ემულატორზე.

## დიზაინი
აპლიკაციას აქვს ერთიანი "teal and mint" ფერების პალიტრა მომრგვალებული კუთხეებითა და ბარათებზე დაფუძნებული განლაგებით, რაც უზრუნველყოფს სუფთა და ინტუიციურ ინტერფეისს.
